策略模式就像是在玩游戏时可以选择不同的角色来使用不同的技能一样。假设你在玩一个游戏，你可以选择成为战士、法师或者射手。每个角色都有自己独特的技能，比如战士可能有近身攻击，法师可以施放魔法，射手可以远程攻击。这就是策略模式的概念：你可以在不同情况下选择不同的“策略”（角色），以便根据需要使用不同的“技能”（方法）。这让你可以根据情况来灵活地做出不同的选择，就像在游戏中切换角色一样
